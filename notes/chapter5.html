Functions and Recursion

Sometimes our program gets bigger in size and it's not possible for a programmer to track which peice of code is doing what.
Function is way to break our code into chunks so that it is possible for a programmer to reuse it.

What is Function?

A function is a block of code which performs a particular task.
A function can be reused by the programmer in a given program any number of times.
NOTE:
All code of this chapter are in chapter_05== (in code) file.
There are some parts of function and code as follows:
1.Function prototype: It is a way to tell the compiler about the function that we are going to define in the program.Here the void indicates that the function returns nothing.
2.Function call: It is a way to tell the compiler to execute the function body at the time the call is made.
NOTE: That the program execution starts from the main function in the sequence the instructions are written.
Function definition: This part contains the exact set of instructions which are executed during the function call in main body.When it is called from main(), the main function falls asleep and gets temporarily suspended.During this time the control goes to the function being called.When the function body is done executing main() resumes.

Important points
Execution of a C program starts from main().
A C program can have more than one function.
Every function gets called directly or indirectly from main().
There are two types of function in C.

Types of Function

1.Library functions: Commn;y required functions grouped together in a library.Like-- printf,scanf,etc.
2.User-defined function: These are the functions declared and defined by the user.

Why Use Function?

1.To avoid rewriting the same logic again and again.
2.To keep track of what we are doing in the program.
3.To test and check logic independently.

Passing Values To The Function

The function which takes values is called parameter and the one which gives values is called return value.
NOTE:
1.Parameters are the values or variables placeholders in the function definition.
Arguments are the actual values passed to the function to make a call.
A function can return only one variable at a time.
If the passed variable is changed inside the function, the funnction call doesn't chnage the value in the calling function.

Recursion

A function defined in C can call itself.This is called recursion.
A function calling itself is also called recusive function.
Important Notes:
1.Recursion is sometimes the most direct way to code an algorithm.
2.The condition which doesn't call the function any further in a recursive function is called the base fuction.
Sometimes, due to a mistake made by a programmer, a recursive function can keep running without returning result in a memory error.
