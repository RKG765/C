Arrays

An array is a collection of similar elements.
Syntax:
int marks[90,123,1223]; ----> Integer Array
char name[]; ----> Character Array or String.
float percentile[90]; ----> Float Array.
NOTE: The array index starts with 0 to (n-1) where n is number which you assigned for making array.
Accessing Elements:
scanf("%d"),&marks[0]; ----> Input first value.
printf("%d",marks[0]);

Pointer Arithmetic

A Pointer can be incremented to point to the next memory location of that type.
If you do one time increment in int,float it will add 4-bytes to the address and you can add so on.In char it will 1-byte for capital and small letters.
Following operations can be performed on pointers in arrays.
1.Addition of a number to a pointer.
2.Subtraction of a number from a pointer.
3.Subtraction of one pointer from another.
4.Comparision of two pointer variables.
