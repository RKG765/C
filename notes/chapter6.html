Pointers

A pointer is variable which stores the address of another variable.Like we used the & means address of in scanf function.
Address- In memory of computer every body have a address.
Like: read the example we make two variables a,b.
Then we store int a=23; and it's address may be 234543.Now we make second variable b=234543. Here we can see that b is pointing towards a because we put a's address into b.So, b will be called Pointer.

& Operator

The address of operator is used to obtain the address of a given variable.
Now we use %u to print &.
The value of address operator (*) is used to obtain the value present at a given memory.

How to declare a pointer?

A pointer is declared using the following syntax.
int *a; Here declared a variable a in type int-pointer.
a=&b; Here store address of b in a.
There are three pointer type declareation.
1. int *a; ----> Pointer to integer.
2. char *a; ----> Pointer to character.
3. float *a; ----> Pointer to float.

Pointer To A Pointer

Just like above a points to b or storing the address of b, we can have another variable k which can further store the address of a.
The type k will be written as below:
int **k;
NOTE:
You can create unlimited numbers of pointer by adding the * before the variable like to store the address of k we will create a new pointer named l.
int ***l;
Write the condition(l=&k;).

Types Of Function Calls

Based on the way we pass arguments to the function,Function calls are two of two types.
1.Call by Value: Here the value of argument are passed to the function.(example in the code.)
In C we usually use a call by value function.
2.Call by reference: Here address of the variables is passed to the function as arguments.
Now the adderesses are passedd to the function, the function can now modify the value of a variable in calling function using * and & operators.
