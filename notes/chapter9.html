Structures

Arrays and Strings can take only similar data like int,char,float at one time.
Structures can hold disimialr data.
Synatax for creating structures is in programs.
So a structure in C is a collection of variables of different types under a single name.
Quick-Quiz:
Write a program to store the details of 3 employee from user defined data.

Why Use Structures

We can create the datatype in the employee structure seperately but when the number of properties in a structure increases,it becomes difficult for us to create data variables without structures.
i.)Structures keeps the data organised.
ii.)Structures makes data management easy for the programmer.

Arrays In Structure

Just like an array of integers,floats or character, we can create an array of structures.
Initialinzing structures: Code are in programs.

Structures In Memory

Structures are stored in continious memory locations.For the structure e1 of type struct employee.
In an array of structures, these employee instances are stored adjacent to each other.Like in case of e1,e2 these will stored as adjacent to each other.

Pointers To Structures

Code in program.
Arrow Operator: Instead of writing (*ptr).name, we can use arrow operator to access structures properties as follows.
(*ptr).name || ptr->name
Here, -> is known as the arrow operator.

Passing Structures To A Function

A structure can be passed to a function just like any other datatype.
Quick-Quiz:
Write a function with structure to display the content in it.

Typedef Keyword

We can use typedef keyword to create an alias name for data types in C.
typedef is more commonly used with structures.
It is also used for making custom datatypes in C.
