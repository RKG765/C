File I/O

The Random Access Memory is volatile and it's content is lost once the program terminates.In order to persists the data we use files.
A file is data stored in storage device.
A C prgram can talk to the file by readint contents and writing contents to it.

File Pointer

The "FILE" is a structure which needs to be created for opening the file.
A FILE pointer is a pointer to this structure of the file.

Writing To A File

We can write to a file in a similar manner like reading.

fgetc() and fputc() Function

fgetc and fputc are used to read and write a character from/to a file.

EOF: End Of File

fgetc returns EOF when all the characters from a file has been read.
