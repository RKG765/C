What is Programming?

Computer programming a medium for us to communicate with computers.It is a way for us to deliver our instructions to the computer.

What is C?

C is a programming language.
It is one of the oldest and finest programming language.
It was developed by Dennis Ritchie at AT & T's Bell labs, USA in 1972.

Uses of C

It is used to program wide variety of systems.
Some of the uses of C are as follows:
1.Major parts of windows, linux and other operating systems are written in C.
2.It is used to write drivers programs for devices like Tablets,Printers,etc.
3.It is used to program embedded systems where programs need to run faster inn limited memory(Microwave,Cameras,etc.)

Variables,Constant&Keywords

Variables

A variable is a container which stores a 'value'.It's value can be changed.
Such as:
a=3; (where a is assigned to 3.)
b='Rich';

Rules for naming variables in C

1.First character must be an alphabet or underscore(_).Like: int tom; & int _tom;
2.No commas(,),blanks allowed.Like: int to m; & int to,m.
3.No special symbol other than (_) allowed.Like: int %tom; & int tom&;
4.Variable names are case-sensitive.Means lowercase and uppercase matters.Like: int tom; & int Tom and soon.
You should make variable names neat and clean.

Constant

An entity whose value doen't change is called a constant.

Types of Constant

Integer constant:They are real number.Like: 1,-234,3442,etc.
Real constant:These are decimal number.Like:2.0,123.23,234.53,etc.
Character constant: These are for character.Like: 'a','@','&','$','etc.'NOTE: If you want to make Character constant then use it within the ''.

Basic Structure of a C program

All C programs have to follow a basic structure.
A C program starts with a main function and execute intruction present inside it.Each instruction is terminated with semicolon(;).

Rules

There are some rules which are applicable to all C programs.
Every program's execution starts form main() function.
All statements are terminated with semi-colon.
Instruction are case-sensitive.
Instructions are executed in the same order in which they are written.

Comments

Comments are used to clarify something about the program in plain language It is a way for us to add notes to our program language.
Single-Line comment: //This is a comment.
Multi-Line comment: /* This is comment.*/.
Comments are not executed and ignored in C Language programs.

Compilation and Execution

A compiler is a computer program which converts a C program into machine language so that it can be easily understood by the computer.
A C program is written in plain text.
This plain text is compilation of instruction in a particular sequence.The compiler performs some basic checks and finally converts the program in to executable.

Library Functions

C Language has a lot of valuable library functions which is used to carry out certain tasks for instance printf function is used to print values on the screen.Such as:
printf("This is %d",i);
%d is for integers.
%f for realvalues.
%c for characters.

Recieving input from user

In order to take input from the user and assing it to a variable, we use scanf function.
syntax for using scanf function:
scanf("%d",&i);
Here & is the 'address of' operator and it means that the supplied value should be copied to the address which is indicated by the variable i.
